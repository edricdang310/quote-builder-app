Hướng Dẫn Vận Hành Hệ Thống

1. Khởi chạy & Thiết lập

Ứng dụng chạy trên cổng mạng mặc định 8080. Bạn có thể vận hành cục bộ bằng Python hoặc triển khai thông qua Docker container:

  • Local Python: Chạy lệnh python server.py.
  • Docker: Chạy lệnh docker compose up --build -d để khởi động chạy ngầm.

2. Cơ chế Lưu trữ & Chia sẻ

Khi bạn nhấn nút "Chia sẻ" hoặc "In/Xuất PDF", toàn bộ cấu hình thô được lưu trữ dưới định dạng JSON siêu nhẹ trên ổ đĩa cứng của máy chủ (trong thư mục Service_quote/). Khi khách hàng xem link chia sẻ, hệ thống sẽ sử dụng lõi kết xuất động để dựng giao diện đẹp mắt trong thời gian thực.

3. Cách in PDF chuẩn A4 (1 trang duy nhất)

  1. Tại cửa sổ thiết lập in của trình duyệt, chọn Save as PDF (Lưu dưới dạng PDF).
  2. Ở phần Cài đặt nâng cao (More Settings), hãy BỎ TÍCH ô Headers and footers (Tiêu đề đầu trang và chân trang).
  3. Hệ thống đã tự động định dạng kích thước A4, tên tệp tin tải về sẽ được tự động gợi ý theo Số báo giá (ví dụ: BG-EVG-2026-06-001.pdf).

4. Quản lý Lịch sử & Phục hồi dữ liệu

Truy cập vào mục Lịch sử để xem thống kê tổng số lượng file. Bạn có thể nhấn "Sửa" để mở lại bất kỳ cấu hình báo giá cũ nào và cập nhật thông số nhanh chóng.
